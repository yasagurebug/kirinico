# Kirinico

- Kirinico は、単色背景から前景を切り抜いて、透過 PNG を作るための Windows デスクトップツールです。
- アンチエイリアスで輪郭が背景と融合してしまっていたり、MPEG/JPEGノイズで輪郭が汚れている画像に特化しています。

## 特徴

- `手動seed` / `色域指定` の 2 モードによる背景指定
- `CF` / `KNN` 切替対応の PyMatting ベース alpha 推定
- despill による後補正
- 出力サイズ変更
- 任意色・任意太さの縁取り
- 設定 JSON のエクスポート / インポート
- 複数画像のドラッグアンドドロップ一括処理

## 切り抜きTips

- **注意** 初回の処理はPyMattingの初期化が行われるため時間がかかります（～数分）
- まずは設定はそのままで結果を見てみます。
- trimapは、「絶対に背景の場所は黒」「絶対に前景の場所は元の絵」として、輪郭がすべてその中間（unknown領域とよびます。赤色がつきます）に含まれるようにするのがコツです。
- unknown領域をPyMattingが推測します。
- trimapに輪郭が含まれているのに結果で透明になってしまう場合は、輪郭推測方法を「精密」にすると改善するかもしれません（ただしすごく重いです）
- 背景色と前景色の差が大きな画像（例：白地に黒主線）は、輪郭設定方法を「色域」にして、主線ギリギリまでunknownが含まれるようにするのがコツです
- 背景にも前景にも同じ色があるような場合（例：白地にドーナツ型で、主線ははっきりしているが、前景にも白が使われている）は、指定モードを「手動seed」にして、背景をクリックしていってください。

## UI

### 背景指定

- `指定モード`
  - `手動seed`: クリックした場所を背景アンカー（seed）として使います。
  - `色域指定`: 指定した背景色に近い全画素を背景候補として使います。
- `背景色`
  - `色域指定` モードで使う背景色です。
  - テキスト、カラーピッカー、スポイトで指定できます。
- `seed 追加`
  - `手動seed` モードで背景を追加します。
- `背景 seed 管理`
  - 追加済み seed の一覧を開きます。

### メイン処理

- `背景許容色差`
  - 背景として確定する色差の広さです。
- `輪郭設定方法`
  - `幅` または `色差` を選びます。
- `輪郭許容色差`
  - `輪郭設定方法 = 色差` のときだけ表示されます。
- `最大輪郭幅`
  - `輪郭設定方法 = 幅` のときの unknown 帯の上限幅です。
- `ノイズ軽減`
  - 背景マスクの小ノイズや小穴埋めの強さです。
- `輪郭推測方法`
  - `高速 = CF`
  - `精密 = KNN`　注意：非常に重いです

### 後補正

- `透明部カット`
  - 低 alpha を切り捨てます。
- `透明部切り上げ`
  - 高 alpha を `1.0` へ切り上げます。
- `輪郭拡張`
  - `0 < alpha < 1` の帯を外側へ何 px 広げて despill（輪郭色補正） 対象に含めるかを決めます。
- `輪郭補正強度`
  - despill の強度です。通常あまり触りません。
- `輪郭明度補正`
  - despill の明度です。輪郭を暗くしたい・明るくしたい場合に触ります。

### 出力サイズ

- `リサイズ方式`
  - `最近傍 / 線形 / Cubic / Lanczos / Area`
  - 通常はLanczosで良いと思います。縮小するときはAreaの方が自然な結果が得られる場合があります。
- `倍率`
- `幅`
- `高さ`

いずれかを変更すると、縦横比を保ったまま残りの値も連動して更新されます。

### 縁取り

- GroupBox 見出し右のチェックで ON/OFF
- `色`
- `太さ`

### 上部ツールバー

- `A_raw保存`
  - matting 後の生 alpha マスクを `<元画像名>_alpha_raw.png` として保存します。
- `再処理`
  - 強制的に画像の再生成を行います。
- `自動再処理`
  - チェックすると、パラメータ変更時に自動的に画像の再生成を行います。
- `高速プレビュー`
  - チェックすると、pyMattingを低解像度で高速に行いプレビューし、その後に実解像度で上書きする動作を行います。
- `ズーム`
- `設定エクスポート`
  - 現在の設定をJSONファイルとしてエクスポートします。
- `設定インポート`
  - JSONファイルを読み込んで設定を復元します。

## 動作前提

- Windows
- .NET 8 SDK
