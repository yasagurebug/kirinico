# Kirinico

Kirinico は、単色背景付きのキャラクター画像から透過 PNG を作るための Windows デスクトップツールです。

## Features

- `手動seed` または `色域指定` による背景指定
- 粗い前景マスクからの内側輪郭帯 alpha 推定
- 前景色の逆算復元
- premultiplied alpha リサイズ
- 任意色・任意太さの縁取り
- 複数画像のドラッグアンドドロップ一括処理

## Settings

### 設定Tips

- 抜きの強さと、輪郭幅はトレードオフに近い関係です。抜きが弱い場合は輪郭幅を大き目に取ると背景色のが減少します。
- ただし、輪郭幅が大きすぎると、内部の色まで半透明になったりする可能性が上がります。
- 一方で、抜きを強くしすぎると、輪郭が失われる可能性が上がります。
- そのため、抜きの強さと輪郭幅の調整で、ちょうどいい位置を見つけるのが重要です。
- 失われた輪郭の補強のために、主線色で縁取りを1px程度つけるのも有効です。


### 背景指定

- `指定モード`
  - `手動seed`: クリックした seed を起点に、背景色に近い連結領域を背景として扱います。ドーナツ状の穴や閉領域内部を背景にしたいときに向いています。
  - `色域指定`: seed を無視し、指定した背景色に近い全画素を背景として扱います。単色背景を一括で抜きたいときに向いています。
- `背景色`
  - `色域指定` モードで使う背景色です。テキスト、スポイト、カラーピッカーで指定できます。画像読込時は左上 `(0,0)` ピクセルの色が初期値になります。
  - テキスト欄の横に現在色プレビューを表示し、背景色スポイト中はホバー直下の色で一時的に更新します。
- `seed`
  - `手動seed` モードのときだけ使えます。
  - `追加`: 元画像または結果画像を 1 回クリックして background seed を追加します。追加後は自動で移動モードへ戻ります。
  - `削除`: seed 一覧ウィンドウを開きます。各 seed のプレビューと座標を見ながら削除できます。

### 処理強度

- `抜き`
  - 背景色として許容する色差の強さです。上げるほど背景として扱う範囲が広がります。
- `ノイズ除去`
  - 背景領域の小穴埋めと微小ノイズ除去の強さです。上げるほど背景 mask を強めに整形します。
- `輪郭幅`
  - 最大何 px まで輪郭を削ってよいかを表します。境界近傍の soft alpha 推定帯の幅として使われます。
- `主線指定`
  - 代表的な主線色です。輪郭の色復元で参照します。
  - テキスト、スポイト、カラーピッカーで指定できます。
  - テキスト欄の横に現在色プレビューを表示し、主線色スポイト中はホバー直下の色で一時的に更新します。
  - 未指定なら、背景色から最も離れた局所前景色だけで復元します。
  - `F_local` が見つかった輪郭帯では、その発見位置までの外側側にだけ影響し、その内側では色復元を行いません。
  - `F_local` が見つからない輪郭帯では、RGB は観測色のまま保持されます。
  - 最終的な輪郭整形では、中間 alpha を持つ画素だけ RGB と alpha を一緒に平滑化します。

### 出力サイズ

- `リサイズ方式`
  - 出力リサイズ時の補間方式です。
  - `最近傍`: ドット感を保ちます。
  - `線形`: 軽く滑らかです。
  - `Cubic`: 線形より少しシャープです。
  - `Lanczos`: 最もシャープ寄りです。既定値です。
  - `Area`: 主に縮小時に安定しやすい方式です。
- リサイズ関連と縁取り関連の変更は、前段の抽出結果を再利用して高速にプレビュー更新します。
- `倍率`
  - 元画像に対する出力倍率です。
- `幅`
  - 出力画像の幅です。
- `高さ`
  - 出力画像の高さです。

いずれかを変更すると、縦横比を保ったまま残りの値も連動して更新されます。

### 縁取り

- `有効 / 無効`
  - 縁取りを出力するかどうかを切り替えます。
- `色`
  - 縁取り色です。テキスト、スポイト、カラーピッカーで指定できます。
  - テキスト欄の横に現在色プレビューを表示し、縁取り色スポイト中はホバー直下の色で一時的に更新します。
- `太さ`
  - 縁取りの太さです。小数指定ができ、`alpha > 0.5` の輪郭から指定太さ分は不透明、その外側 `1.0px` にだけアンチエイリアス帯を付けて出力されます。

## Toolbar

- `自動再処理`
  -  ON のときは設定変更や seed 編集のたびに自動で再処理します。OFF のときは `再処理` ボタンを押すまで結果を更新しません。
- `設定エクスポート / 設定インポート`
  - 現在の設定を JSON として保存、または読み込みます。
- `About`
  - exe 名、バージョン、作者名、GitHub リンクを確認できます。
